# Beads Manager
Use the `bd` executable to read and write tasks.

## Tools
- `bd ready`: Get the next available task (high priority, unblocked).
- `bd create <title>`: Create a new task.
- `bd close <id>`: Mark a task as complete.
- `bd structure`: View the full project graph.
